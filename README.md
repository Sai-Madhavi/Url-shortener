# 🔗 URL Shortener Web Application

A Flask-based web application that converts long URLs into short, shareable links and redirects users efficiently using a lightweight SQLite database.

## 🚀 Features
- Generate unique 6-character short URLs  
- Redirect short links to original URLs  
- Store URL mappings using SQLite  
- Simple and user-friendly interface  
- Runs locally on browser  

## 🛠 Technologies
Python, Flask, SQLite, HTML, CSS  

## ▶ How to Run
pip install flask  
python app.py  
Open: http://127.0.0.1:5000  

## ⚙️ Working
User enters a URL → system generates a short code → stores it in database → displays short link → accessing it redirects to original URL

Add authentication, click tracking, custom URLs, and cloud deployment
